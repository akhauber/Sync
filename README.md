# Sync (iOS SwiftUI Prototype)

Sync is a mobile-first shift scheduling concept app built from the provided requirements document.

## Included
- Role-based login (Worker / Manager)
- Worker tabs: My Schedule, Marketplace, Availability, Profile
- Manager tabs: Schedule, Manage, Notifications, Profile
- Design system primitives: glass cards, inset wells, soft buttons, filter chips, emoji mood scale
- Manager insights view with calm card-based reporting
- Marketplace claim/sell concept actions
- Post-shift check-in sheet

## Notes
This repository contains a SwiftUI prototype scaffold intended to be used inside an Xcode iOS app target.
